# Tight Bounds on the Rainbow Connection Number of Random Regular Graphs

Mathematics research project generated by NeuriCo on 2026-03-08.

## Overview

This repository contains a mathematical research project investigating the rainbow connection number rc(G) of random d-regular graphs G(n,d).

**Central conjecture**: rc(G(n,d)) = (1+o(1)) * log(n)/log(d-1) w.h.p.

This would mean the rainbow connection number equals the diameter asymptotically.

## Key Results

- **Lower bound** (proved): rc(G(n,d)) >= (1+o(1)) * log(n)/log(d-1)
- **Upper bound** (proved, following Kamcev-Krivelevich-Sudakov 2015): rc(G(n,d)) <= (2+o(1)) * log(n)/log(d/2-1) for d >= 6
- **Computational**: rc(G) = diam(G) in 72.5% of small cases tested; always equal for n <= 10

## Mathematical Setup

The **rainbow connection number** rc(G) of a connected graph G is the minimum number of colors k such that the edges of G can be colored with k colors so that every pair of vertices is connected by a rainbow path (a path with all distinct edge colors).

For a random d-regular graph G(n,d):
- rc >= diam(G) always
- diam(G(n,d)) = (1+o(1)) * log(n)/log(d-1) w.h.p. [Bollobas-de la Vega 1982]
- rc(G(n,d)) = O(log n) [Dudek-Frieze-Tsourakakis 2014 for d >= 4; Molloy 2017 for d=3]
- rc(G(n,d)) <= c*log(n)/log(d) for d >= 5 [Kamcev-Krivelevich-Sudakov 2015]

## Proof Techniques

1. **Lower bound**: rc >= diam (trivial) + diam ~ log(n)/log(d-1) (Bollobas-de la Vega)

2. **Upper bound (edge-splitting)**:
   - Lemma: rc(G) <= diam(G1) + diam(G2) when G = G1 + G2 edge-disjointly [KKS 2015]
   - By contiguity: G(n,d) ~ G(n,d1) + G(n,d2) with d1+d2=d [Robinson-Wormald 1994]
   - Each factor has diameter ~ log(n)/log(di-1)
   - Sum gives the upper bound

3. **Potential tight bound approach** (open):
   - Show a random coloring with (1+eps)*log_{d-1}(n) colors is rainbow w.h.p.
   - Uses: expander has many vertex-disjoint short paths between any pair
   - Hard part: making the path count precise enough for a union bound

## Open Problems

1. **Main conjecture**: Is rc(G(n,d)) = (1+o(1)) * log(n)/log(d-1)?
2. **Additive constant**: Is rc(G(n,d)) <= diam(G(n,d)) + O(1)?
3. **Growing d**: What is rc when d = d(n) grows with n?
